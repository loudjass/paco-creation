# Paco-Creations 🧶

Bienvenue sur le projet **Paco-Creations**, un site vitrine élégant et artisanal pour présenter des créations uniques faites main (crochet, accessoires, poupées).

## 🚀 Fonctionnalités
- **Design Artisanal Premium** : Palette de couleurs chaleureuse (Beige, Terre cuite, Rose poudré).
- **Entièrement Responsive** : S'adapte parfaitement sur mobile, tablette et desktop.
- **Animations Fluides** : Effet d'apparition au défilement pour une expérience premium.
- **Formulaire de Contact** : Système simple avec `mailto` pour recevoir des demandes directement par email.

## 📁 Structure du Projet
- `index.html` : Structure principale avec SEO optimisé.
- `styles.css` : Design system complet et layout moderne.
- `script.js` : Interactivité et gestion du formulaire.
- `public/images/` : Dossier contenant les photos des créations.

## 🛠️ Installation Locale
1. Clonez ce dépôt.
2. Ouvrez `index.html` dans votre navigateur préféré.
3. C'est tout ! Pas de serveur de build requis (HTML/CSS/JS pur).

## 🌍 Déploiement

### Vers GitHub
1. Créez un nouveau dépôt sur GitHub.
2. Initialisez git dans votre dossier local :
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/votre-compte/paco-creations.git
   git push -u origin main
   ```

### Vers Vercel (Recommandé)
1. Connectez-vous sur [vercel.com](https://vercel.com).
2. Cliquez sur **"Add New"** > **"Project"**.
3. Importez votre dépôt GitHub `paco-creations`.
4. Vercel détectera automatiquement le projet statique. Cliquez sur **"Deploy"**.
5. Votre site est en ligne en quelques secondes !
